什么是恩金币？

恩金币是一种ERC20代币，拥有由开源智能合约、软件开发工具包（SDK工具包）和应用组合而成的强大框架支持。

管理区块链里的虚拟商品

这是一个去中心化的虚拟道具创造、管理、分发和交易平台。给予游戏开发商和内容创造者新的虚拟所有权模式。

配备专属游戏行业的SDK工具包

恩金币平台专为游戏行业服务。支持多语言、多钱包和多支付平台的SDK工具包即将面世，且完全开源。

拥有真正现金价值的游戏体验

线上玩家可以跨游戏、跨社区地转移或交易资产，实现游戏过程中数字钱包的真实价值增长。

提供诈骗和高额收费的解决方案

每1次合法虚拟商品购买，就有7.5件商品因诈骗而损失。恩金币铸造的虚拟商品将确保交易的可信度和安全度，同时只收取最低交易手续费。

创建&管理虚拟商品

以恩金币为母币铸造独特的游戏商品、货币及虚拟代币。此类资产可随时被兑换回恩金币。

内容创建者的软件

将开发针对铸造智能合约的 网页、移动应用程序和基于 API 的用户界面，使资产创建和管理过程变得简单。

去中心化支付网关

使用开源 SDK 工具构建自定义支付网关，并接受恩金币和所有自定义币。
